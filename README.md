# ⚙️Process Scheduling Visualizer
The project has been designed to simplify the learning of these concepts by providing an interactive platform where users can input process details and observe how different scheduling algorithms operate. By simulating algorithms such as First Come, First Serve (FCFS), Shortest Job First (SJF), and Round Robin (RR), the project demonstrates differences in execution order. It highlights each algorithm's impact on process efficiency.

## ⚙️ Key Features of Website:-

🔹 User-friendly interface to enter process details (Process ID, Arrival Time, Burst Time)  
🔹 Supports multiple scheduling algorithms:- FCFS (First Come First Serve), SJF (Shortest Job First) and Round Robin
🔹 Dynamic visualization of process execution and scheduling order  
🔹 Real-time output display for better understanding of algorithm behavior  
🔹 Structured layout using semantic HTML and styled with CSS for clarity  

## 💻 Technologies Used:-

• **HTML** – Provides the structural foundation for content organization.  
• **CSS** – Styles the layout, ensuring a visually appealing presentation.  
• **JavaScript** – Implements scheduling logic and interactive visualization.
